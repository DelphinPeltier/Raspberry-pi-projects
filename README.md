# raspberry-pi-projects

## Description

Ce repository regroupe plusieurs projets réalisés avec un Raspberry Pi dans le cadre d’expérimentations personnelles autour des systèmes, du réseau et de l’automatisation.

---

## Projets réalisés

### Serveur NAS

Configuration du Raspberry Pi comme serveur de stockage réseau permettant de partager des fichiers sur le réseau local et distant.

Fonctionnalités :
- partage de fichiers
- accès depuis plusieurs appareils
- stockage centralisé

---

### Serveur multimédia

Mise en place d’un serveur multimédia permettant d’accéder à des contenus vidéo et audio depuis différents appareils.

Fonctionnalités :
- lecture de contenus multimédia
- accès depuis le réseau local et distant
- gestion d’une bibliothèque multimédia

---

### Home Assistant

Mise en place d’un serveur domotique permettant de centraliser et automatiser différents équipements connectés.

Fonctionnalités :
- gestion d’équipements domotiques
- automatisation de scénarios
- contrôle centralisé

---

### n8n

Utilisation de n8n pour automatiser différentes tâches et créer des workflows entre plusieurs services.

Fonctionnalités :
- automatisation de tâches
- création de workflows
- intégration de différents services

---

### Android Auto Raspberry Pi

Création d’un système embarqué utilisant un Raspberry Pi et un écran pour afficher une interface Android Auto dans une voiture.

Objectifs :
- interface multimédia embarquée
- affichage sur écran dédié
- expérimentation d’un système embarqué

---

## Technologies utilisées

- Raspberry Pi
- Linux
- Réseau
- Services serveur
- Domotique

---

## Objectif

Ces projets ont pour but d’approfondir mes connaissances en systèmes, réseau et administration Linux dans un environnement de laboratoire personnel (homelab).
